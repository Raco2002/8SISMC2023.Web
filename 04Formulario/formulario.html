Todos los formularios que se manejan dentro de la web utlizan metodos de envio para cada uno de los datos, que nos sirven para las solicitudes de informacion (request) los mas comunes son:
Get: Este metodo se encarga de enviar la informacion por la url o la salida estandar del navegador, esto propicia que se pueda visualizar toda la informacion del formulario en la url.
POST: Este metodo se encarga de enviar la informacion por url, pero de forma oculta, lo que ofrece cierto criterio de seguridad, este metodo se usa comunmente para insertar un nuevo registro dentro de los sitemas de informacion.
